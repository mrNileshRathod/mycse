* Tasks to be performed :

1.Cross Join

 mysql> SELECT CName, OID FROM CUSTOMER CROSS JOIN C_ORDER;

2.Inner Join

 mysql> SELECT CUSTOMER.CName, C_ORDER.OID FROM CUSTOMER INNER JOIN C_ORDER ON CUSTOMER.CID = C_ORDER.CID;

3.Left Join

 mysql> SELECT CUSTOMER.CName, C_ORDER.OID FROM CUSTOMER LEFT JOIN C_ORDER ON CUSTOMER.CID = C_ORDER.CID;

4.Right Join

 mysql> SELECT CUSTOMER.CName, C_ORDER.OID FROM CUSTOMER RIGHT JOIN C_ORDER ON CUSTOMER.CID = C_ORDER.CID;
